# Dishcovery

deployed url: https://dishcovery-git-vue-version-solagratia7s-projects.vercel.app?_vercel_share=114BISAsoDdtB132ltqAdWnhvsfFR1xa

Dishcovery is a recipe discovery web app built with Vue 3 and Vite.  
It allows users to search for recipes, manage ingredients, explore cooking steps, and track nutritional intake in a clean, responsive interface.

## Installation

```bash
git clone https://github.com/SolaGratia7/Dishcovery.git
cd source
npm install
npm install vue-router
npm install dayjs
git checkout vue-version
npm run dev
```
